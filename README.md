# Go 从 0 学习

按阶段学 Go 1.26，每课三件事：理解概念 → 跑通代码 → 做练习。

## 运行方式

```bash
# PATH 需包含 %USERPROFILE%\go-install\go\bin
go run ./lesson-01/
go run ./lesson-02/
# ...以此类推
```
